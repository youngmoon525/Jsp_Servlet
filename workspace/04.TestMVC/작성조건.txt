○ tblBook 테이블 생성 : com.hanul.study package → tblBook.sql 저장
	- 도서명	title		varchar2(30)
	- 저자		name		varchar2(20)
	- ISBN		isbn		varchar2(20)	primary key not null
	- 출판사	company		varchar2(20)
	- 단가		cost		number
	- 수량		qty			number
	- 가격		price		number

○ bookMain.html(메인화면) : WebContent → 도서관리 예시화면.hwp
	- 모든 항목은 반드시 입력되어야 한다.
	- ISBN은 유효성 검사를 수행 : xxx-xx-xxxx-xxx-x ▶ 정규식 : javascript, jQuery
	- 단가와 수량은 숫자만 입력되어야 한다.
	- 위의 작성조건을 만족하면 서버로 전송 → Controller(Servlet)
	
○ Servlet 정의 : Java Resource > src > default package ▶ web.xml : servlet-mapping 수동 설정
	- InsertServlet.java(is.do) : 도서정보 입력
	- GetAllListServlet.java(list.do) : 전체도서 목록보기
	- DeleteServlet.java(ds.do) : 도서정보 삭제
